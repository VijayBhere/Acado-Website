Powerful features for better connectivity.

Schools use Acado to simplify the daily activities of admins and teachers in school and for better communication of everything that matters to the parents with ease.

User management

Create new users, update the details of existing users, add or move students accross classes, etc. everything at one place.

Contact details of every user are stored in Acado therefore you can access it anywhere on the go, this helps you keep your personal contact list in phone unaffected.

Users get notified of their account creation via SMS which you can also choose to send at the time of account creation or later.

User data security

User data stored in Acado is safe as data access inside the app is rectricted and sensitive information like phone numbers of other users are not accessible by parents or students.

Document storage

School admins or teachers can upload user related documents in the form of scanned images or files. These files can be accessed by the school or only by the respective student.

Upload user list

Creating one user at a time can be quite time-consuming and frustrating. To make that easy, Acado allows you to upload a list of users.

After you upload a list of users, Acado creates user accounts for each entry in the file and sends them invitation SMS in just a single click.

User groups

User groups allow school admins and teachers to create groups of users by adding some users from different classes or by selecting multiple classes all together.

These groups can be reused across modules and provides you the flexibility to share or publish data like fees, schedule etc. to a group of users.

Fees

Quickly create and publish fees to a group of users that you had created in user groups module. You can also enforce a late fee for payments after the given due date.

Payments can be done online as well as manually. For manual payments, admin can mark the fees as paid, using the admin panel.

Refunds

Refunding a payment is also possible with Acado. For the payments done online, you can refund through app and for the manual payments, you can mark the refund status for the student.

Unmatched Security

Each activity done on a particular fees is tracked, therefore, there is no way any unauthorized activity goes unnoticed.

Bulletin

A bulletin board for sharing achievements and broadcasting other notices to all.

Your school team has won an inter-school cup, you have equipped classes with smart teaching programs, etc. now can be shared with everyone in the school.

Messages

Messages is an inbuilt chat feature to enable guaranteed message delivery, schools for faster communication with their staff or parents.

Now you can chat with anyone in the school using the messages feature in Acado without having to share your contact details.
