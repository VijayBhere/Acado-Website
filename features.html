Powerful features for better connectivity.

Acado provides a complete set of features to support your day to day activites at school.

User management

Create new users, update the details of existing users, add or move students accross classes, etc. everything at one place.

Contact details of every user are stored in Acado therefore you can access it anywhere on the go, this helps you keep your personal contact list in phone unaffected.

Users get notified of their account creation via SMS which you can also choose to send at the time of account creation or later.

User data security

User data stored in Acado is safe as data access inside the app is rectricted and sensitive information like phone numbers of other users are not accessible by parents or students.

Document storage

School admins or teachers can upload user related documents in the form of scanned images or files. These files can be accessed by the school or only by the respective student.

Upload user list

Creating one user at a time can be quite time-consuming and frustrating. To make that easy, Acado allows you to upload a list of users.

After you upload a list of users, Acado creates user accounts for each entry in the file and sends them invitation SMS in just a single click.

User groups

User groups allow school admins and teachers to create groups of users by adding some users from different classes or by selecting multiple classes all together.

These groups can be reused across modules and provides you the flexibility to share or publish data like fees, schedule etc. to a group of users.

Fees

Quickly create and publish fees to a group of users that you had created in user groups module. You can also enforce a late fee for payments after the given due date.

Payments can be done online as well as manually. For manual payments, admin can mark the fees as paid, using the admin panel.

Refunds

Refunding a payment is also possible with Acado. For the payments done online, you can refund through app and for the manual payments, you can mark the refund status for the student.

Unmatched Security

Each activity done on a particular fees is tracked, therefore, there is no way any unauthorized activity goes unnoticed.

Bulletin

A bulletin board for sharing achievements and broadcasting other notices to all.

Your school team has won an inter-school cup, you have equipped classes with smart teaching programs, etc. now can be shared with everyone in the school.

Messages

Send messages to a group of users or a single user using the inbuilt chat feature. It enables faster communication without having to share your phone numbers with anyone.

Delivery and read receipts keep you informed whether the message was delivered and read.

Group chats

Broadcast a message to a group of people by creating chat groups. You can create any number of groups that may include only teachers, only parents or all.

Impose read or write permission so as to restrict people from sending messages. Therefore users with write permission can send messages and all others can only read the messages.

Personal chats

Personal chats facilitates direct message delivery to a person. Teachers can directly communicate with parents about their pupils progress or parents can ask any query to the teacher.

SMS

Inbuilt SMS features lets you communicate with parents who don't use a smartphone or have not yet installed the Acado app on their phone.

SMS can be sent to a group of users using the contact details available in Acado. It won't cost your main account balance to send SMS as they will be sent using our integrated SMS gateway.

Attendance

Mark daily attendance of students. These records can be accessed anytime later by school or parents.

Parents can view the attendance as soon as you mark it, therefore, they can keep track of their childs attendance on a daily basis.

Leaves

Parents can request leaves for their pupils. Teachers get notified of the leave requests and they can either approve or deny the request.

Reports

Generate numerous reports using the available attendance records. Parents can view a monthly summary for their pupil, School can generate an accumulated report for a given time frame.

Exam reports

Easily publish marksheets of a student to their parents by entering marks in Acado.

Grades

Acado also enables you to create a grade system so that you can display grades alonside the marks or just display grades in the marksheet.

Schedules

Easily publish schedules for exams, gatherings, sport days or any other events. Users get notified of all the published schedules.

Surveys

Create surveys to get quick feedback from parents on various things like meal preferences, trip locations, event times, etc.

Get opinions from parents about the new implementations that you have done in school and make them feel involved in the decision making process.
