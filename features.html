Powerful features for better connectivity

Acado provides a complete set of features to support your day to day activites at school.

REGISTER NOW REQUEST A DEMO

User management

Create new users, update the details of existing users, add or move students across classes, everything in one place.

Acado stores contact details of every user, and therefore, you can access it anywhere on the go keeping the personal contact list in device unaffected.

Users get notified of their account creation via SMS which you can also choose to send at the time of account creation or later.

User data security

User data stored in Acado is safe as data access inside the app is restricted and sensitive information like phone numbers of other users are not accessible by parents or students.

Document storage

School admins or teachers can upload user related documents in the form of scanned images or files. These files can be accessed by the school or only by the respective student.

Upload user list

Creating one user at a time can be quite time-consuming and frustrating. To make that easy, Acado allows you to upload a list of users.

After you upload a list of users, Acado creates user accounts for each entry in the file and sends them invitation SMS in just a single click.

User groups

User groups allow school admins and teachers to create groups of users by selecting users from different classes or by selecting multiple classes altogether.

These groups can be reused across modules and provide you the flexibility to share or publish data like fees, schedule, to a group of users.

Fees

Create and publish fees to a group of users that you had created in user groups module. Enforce a late fee for payments after the given due date.

Payments can be done online as well as manually. For manual payments, admin can mark the fees as paid, using the admin panel.

Refunds

Refunding a payment is also possible in Acado. For online payments, refund through the app and for the manual payments, mark the status for the fee as refunded.

Unmatched Security

Each activity done on a particular fee is tracked, and therefore, any unauthorized activity never goes unnoticed.

Bulletin

A bulletin board for sharing achievements and broadcasting other notices to all.

Your school team has won an inter-school cup, you, have equipped classes with smart teaching programs, and everything else can now be shared with everyone in the school.

Messages

Send messages to a group of users or a single user using the inbuilt chat feature. It enables faster communication without having to share your phone numbers with anyone.

Delivery and read receipts keep you informed whether the message was delivered and read.

Group Chats

Broadcast a message to a group of people by creating chat groups. You can create any number of groups that may include only teachers, only parents or all.

Impose read or write permission to restrict people from sending messages. Therefore, users with write permission can send messages and all others can only read them.

Personal Chats

Personal chats facilitate direct message delivery to a user. Teachers can directly communicate with parents, about their child's progress or parents can ask any query to the teacher.

SMS

Inbuilt SMS features let you communicate with parents who don't use a smartphone or have not yet installed the Acado app on their phone.

SMS can be sent to a group of users using the contact details available in Acado and, it won't cost your account balance to send SMS as they will be sent using our integrated SMS gateway.

Attendance

Mark daily attendance of students. These records can be accessed anytime later by school or parents.

Parents can view the attendance as soon as you mark it, therefore, they can keep track of their child's attendance on a daily basis.

Leaves

Parents can request leaves for their pupils. Teachers get notified of the leave requests and they can either approve or deny the request.

Reports

Generate numerous reports using the available attendance records. Parents can view a monthly summary for their child, School can generate an accumulated report for a given time frame.

Exam reports

Easily publish mark sheets of a student to their parents by entering marks in Acado.

Grades

Acado also enables you to create a grading system so that you can display grades alongside the marks or just display grades on the mark sheet.

Schedules

Easily publish schedules for exams, gatherings, sports days or, any other event. Users get notified of all the published schedules.

Surveys

Create surveys to get quick feedback from parents on various things like meal preferences, trip locations, or event times.

Get opinions from parents about the new implementations that you have done in school and make them feel involved in the decision making process.
